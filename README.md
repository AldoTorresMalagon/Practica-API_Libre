# Practica API Libre - Quotes de Anime

Aplicación web que consume la API pública **Animechan** para mostrar frases célebres (quotes) de animes.

## Funcionalidades
- Listado de quotes (por defecto de Overlord)
- Búsqueda por anime o personaje
- Ordenamiento (por anime o personaje, asc/desc)
- Paginación (20 quotes por página)
- Detalle completo al seleccionar una quote

## Tecnologías
- HTML + CSS + JavaScript vanilla
- API: Animechan<a href="https://animechan.io/docs" target="_blank" rel="noopener noreferrer nofollow"></a> – tier free (5 req/hora)

## Notas
- El tier gratuito tiene límite de 5 peticiones por hora. Si ves Error 429, espera 60 minutos.
- Desplegado con GitHub Pages.
